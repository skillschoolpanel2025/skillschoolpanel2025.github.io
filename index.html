Инструкция по использованию SkillSchool Panel

Общая информация

SkillSchool Panel - это административная панель для управления пользователями, просмотра статистики и настройки системы. Панель интегрирована с базой данных Supabase и предоставляет удобный интерфейс для всех административных задач.

Совет: Для входа используйте авторизованную почту (saralashvilischool2025@gmail.com) и установленный пароль. Доступ ограничен только для авторизованных пользователей.

Управление пользователями

Раздел управления пользователями позволяет создавать новые учетные записи различных типов:

  • Ученики - пользователи с ограниченными правами доступа, имеющие баланс
  • Учителя - пользователи с доступом к учебным материалам
  • Администраторы - пользователи с полными правами доступа к системе

Безопасность

Раздел безопасности содержит инструменты для управления вашей учетной записью администратора:

  • Смена пароля - позволяет изменить текущий пароль (требуется знание текущего пароля)
  • Смена почты - изменение email-адреса (требуется подтверждение нового адреса)
  • Журнал входов - просмотр истории авторизаций в систему

Статистика

Раздел статистики отображает графики и таблицы с ключевыми показателями системы:

  • Распределение пользователей - количество пользователей по категориям
  • Бекап данных - возможность скачать резервную копию базы данных

База данных и Сервер

Разделы базы данных и сервера предоставляют инструменты для работы с данными и мониторинга серверной части:

  • Просмотр и редактирование таблиц - прямой доступ к данным
  • Логи сервера - просмотр журналов системы
Внимание! Будьте осторожны при внесении изменений в базу данных. Рекомендуется сделать бекап перед значительными изменениями.

# Инструкции по тестированию Table Editor через консоль (F12)

Вызовите любую из этих команд в консоли браузера:

1. window.testTableError('not_found')
   — Показывает ошибку "Таблица не найдена или не загружена" внутри карточки пользователя.

2. window.testTableError('empty')
   — Показывает сообщение "Нет данных в таблице" внутри карточки пользователя.

3. window.testTableError('server')
   — Показывает ошибку "Ошибка сервера при загрузке данных" внутри карточки пользователя.

4. window.testTableError('loading')
   — Показывает индикатор загрузки внутри карточки пользователя.

5. window.testTableError('success')
   — Показывает статус "Данные успешно загружены" внутри карточки пользователя.

---

**Пример:**

window.testTableError('not_found');

---

Все сообщения будут отображаться внутри карточки пользователя (user-card) с красивым дизайном.
